The positioning scheme used in this website are:

  • CSS Grid has been used to implement the layout of the Information Block. (lines 23-25 in home.css)
  • The header's position has been made fixed and its z-index has been increased to make it hover over the rest of the page (master.css, lines 22-30)

The basic CSS3 techniques used in this website are:

  • Rounded borders have been used on divs and headings on both pages(lines 58 in home.css)(lines 22, 32 & 107 in order.css)
  • Box shadow has been used on the headings 'Sushi', 'Dessert' and 'Drinks' inside the menu div(lines 33 in order.css)
  • Animation is used on several high-level divs, and the images and inputs in the menu-items to make them fade in slowly(lines 54-61 in master.css)

The styling techniques used in this website are:

  • External stylesheet master.css is used for header (and some tags inside header), footer, body, and anchor (linking for header to homepage, undo link) styling which are common to all pages.
  • External stylesheets order.css and home.css are used to provide styling to individual components of the respective pages.
  • The common theme used in both pages is header on top, footer at the bottom, and the content blocks in the middle, arranged in layouts to provide a nice viewing experience.
